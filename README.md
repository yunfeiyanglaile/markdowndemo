## Windows 用户下载指南

如果你使用 **Windows 电脑**，推荐下载以下 Markdown 编辑器：

### Typora（推荐）

Typora 是一款简洁美观的 Markdown 编辑器，支持实时预览。

**Windows 用户请下载：**

- 前往官网：[https://typora.io](https://typora.io)
- 点击 **Download** 页面
- 选择 **Windows** 版本（`typora-setup-x64.exe` 适用于 64 位系统，绝大多数现代 Windows 电脑均为 64 位）

> **如何确认自己的系统是 32 位还是 64 位？**
>
> 右键点击「此电脑」→「属性」→ 查看「系统类型」。
> - 显示 `64 位操作系统` → 下载 `typora-setup-x64.exe`
> - 显示 `32 位操作系统` → 下载 `typora-setup-ia32.exe`

### 其他推荐的 Windows Markdown 编辑器

| 编辑器 | 下载地址 | 说明 |
|--------|----------|------|
| VS Code | [https://code.visualstudio.com](https://code.visualstudio.com) | 免费，安装 Markdown 插件即可使用 |
| Obsidian | [https://obsidian.md](https://obsidian.md) | 免费，支持双向链接，适合知识管理 |
| Typora | [https://typora.io](https://typora.io) | 收费（$14.99 买断），界面简洁，所见即所得 |

## 总结

**对于使用 Windows 电脑的用户：**

- 推荐下载 **Typora Windows 版**（`typora-setup-x64.exe`，适用于 64 位 Windows）
- 如果想要免费方案，推荐使用 **VS Code** + Markdown 插件
